# Installation

## roblox-ts

Commander requires **roblox-ts**. An installation guide can be
found in its [documentation](https://roblox-ts.com/docs/setup-guide).

## Commander

<Tabs>
<TabItem value="npm" default>

```bash title="Terminal"
npm install @rbxts/commander
```

</TabItem>
<TabItem value="Yarn">

```bash title="Terminal"
yarn add @rbxts/commander
```

</TabItem>
<TabItem value="pnpm">

```bash title="Terminal"
pnpm add @rbxts/commander
```

</TabItem>
